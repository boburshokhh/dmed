# Быстрый старт

## 1. Установка зависимостей

```bash
cd front
npm install
```

## 2. Настройка переменных окружения (опционально)

Создайте файл `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000
```

Если бэкенд работает на другом адресе, укажите его здесь.

## 3. Запуск

```bash
npm run dev
```

Откройте браузер: http://localhost:3000

## 4. Проверка работы

1. Введите PIN-код (4 цифры)
2. Нажмите "Открыть"
3. Если PIN правильный, откроется страница с документом

## Примечания

- Убедитесь, что Flask бэкенд запущен на порту 5000 (или укажите другой порт в `.env.local`)
- Если возникают проблемы с CORS, добавьте `flask-cors` в бэкенд
