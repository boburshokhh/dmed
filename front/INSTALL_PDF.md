# Установка библиотеки для рендеринга PDF

## Установка зависимостей

После обновления `package.json`, выполните установку новых зависимостей:

```bash
cd front
npm install
```

Это установит:
- `react-pdf` - библиотека для рендеринга PDF документов
- `pdfjs-dist` - PDF.js от Mozilla

## Запуск приложения

После установки запустите приложение:

```bash
npm run dev
```

## Что было добавлено

1. **Страница просмотра PDF** (`/document/[docNumber]/page.js`)
   - Полноэкранный рендеринг PDF
   - Адаптивная ширина для мобильных устройств
   - Без лишних границ и элементов

2. **Кнопка печати**
   - Расположена внизу справа
   - Открывает диалог печати браузера
   - Работает на всех устройствах (телефоны, планшеты, десктоп)

3. **Настройки Next.js**
   - Добавлены webpack настройки для react-pdf
   - Конфигурация для работы с PDF файлами

## Использование

После ввода правильного PIN-кода на странице `/access/{uuid}`, пользователь автоматически перенаправляется на страницу с PDF документом.

На странице отображается:
- PDF документ в полном размере
- Кнопка печати (синяя круглая кнопка с иконкой принтера внизу справа)

При нажатии на кнопку печати открывается стандартное окно печати браузера.

## Требования

- Node.js 16+
- npm или yarn
- Современный браузер с поддержкой PDF.js
